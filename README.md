# Astro + TailwindCSS project stater

This repository will have minimalistic project Starter. Astro 5 and Tailwind 4. It also includes Alien JS.

## Usage

Checkout and use as project.

```bash
clone
pnpm install
pnpm dev
```

## Includes
* Astro 5.3
* Astro Icon
* Astro SEO
* Astro MDX
  
* Tailwindcss 4

* Alpinejs

* Netlify
